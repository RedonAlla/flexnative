**AvatarGroup** is part of the [**@flexnative/avatar**](../) package.
To use the **AvatarGroup** you needs to install `@flexnative/avatar` package.
[Fallow this link](./index.mdx#installation) on how to install the package.

## API

```jsx
import { AvatarGroup } from '@flexnative/avatar';
```

## Component
A React component that represents a group of avatar items.

**AvatarGroup**
Type: `React.PureComponent<AvatarGroupProps>`

## Props
Properties for the AvatarGroup component.

### AvatarGroupProps

:::info
AvatarGroup extends [`ViewProps`](https://reactnative.dev/docs/view#props) from **React Native**.
:::

|  Name   | Type | Required | Default Value | Description |
| -- | -- | -- | -- | -- |
| sortIndex | <code>'asc' \| 'asc'</code> | **false** | `'asc'` | `asc` or `desc` are supported |
| itemPadding | `number` | **false** | 13 | Padding between avatar items. |
| itemBorderWidth | [`BorderWidth`](../../../theme/default-theme#borderwidth) | **false** | `thick` | Optional avatar borders width. |
| itemBorderColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | `theme.colors.card` | Borders color according `react-native` ColorValue. |
| children | `Array<React.ReactElement<AvatarProps>>` | **true** | `full` | |

    paddingRight: 8,
