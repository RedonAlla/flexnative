**Spinner** is part of the [**@flexnative/icons**](../icons/index.mdx) package.
To use the **FlexNative Spinner** you needs to install `@flexnative/icons` package.
[Fallow this link](../icons/index.mdx#installation) on how to install the package.

You can use any of paskage's icon as apinner.

## API

```jsx
import { Spinner } from '@flexnative/icons';
```

## Component
A memoized React component that renders a spinning icon using animated styles.

**Spinner**

Type: <code>React.MemoExoticComponent\<(props: [IconProps](#properties)) => React.JSX.Element></code>

## Properties

:::info
and `Spinner` components expects props of type [`IconProp`](../icons/basic-usage.mdx#properties).
:::
