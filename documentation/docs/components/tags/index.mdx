The **FlexNative Tags** is part of the `FlexNative` and is available under the `@flexnative/tags` NPM package.

## Dependencies
- [`@flexnative/theme-context`](../../theme/index.mdx)
- [`@flexnative/icons`](../icons/index.mdx)
- [`@flexnative/ui-constants`](../../utilities/ui-constants.mdx)

<Installation title="FlexNative Tags" name='@flexnative/tags' />

## API

```jsx
import Tag from '@flexnative/tags';
```

## Component

**Tag**

Type: <code>React.PureComponent\<[TagProps](#properties)\></code>

## Properties

:::info INFO
`TagProps` props extends [`ViewProps`](https://reactnative.dev/docs/view#props) from `react-native`
:::

|  Name | Type | Required | Default Value | Description |
| -- | -- | -- | -- | -- |
| text | `string` | **true** | `undefined` | Text to display on tag. |
| textColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | `theme.color.default` | Text Color. |
| backgroundColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | `theme.color.default` | Background Color. |
| children  | [`ReactNode`](https://reactnative.dev/docs/react-node) | **false** | `undefined` | ReactNode to render custom react element. |
| borderWidth  | [`BorderWidth`](#borderwidth) | **false** | `'none'` | Optional borders width. |
| borderColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | `undefined` | Borders color according `react-native` ColorValue. |
| onDelete | (event: React.BaseSyntheticEvent) => void | **false** | `undefined` | If provided, will render a Delete icon button, which will call this callback on press. |
| size | [`TagSize`](#tagsize) | **false** | `'default'` | Tag Size variable. |
| type | [`TagType`](#tagtype) | **false** | `'default'` | Tag type. <Admonition type="warning">To have appearance for <code>'ghost'</code>, with custom color you should use hex colors.</Admonition> |
| radius | [`TagRadius`](#tagtype) | **false** | `'medium'` | Tag border radius. |
| color | [`TagColor`](#tagcolor) | **false** | `theme.color.default` | Color by theme or a custom color according `react-native` ColorValue. |

### BorderWidth
One of `'none' | 'hairline' | 'thin' | 'base' | 'thick' | number`

|  Name   | Type | Value |
| ------- | ---- | ----- |
| none | `number` | 0 |
| hairline | [`StyleSheet.hairlineWidth`](https://reactnative.dev/docs/stylesheet#hairlinewidth) | This is defined as the width of a thin line on the platform. It can be used as the thickness of a border or division between two elements. |
| thin | `number` | 1 |
| base | `number` | 2 |
| thick | `number` | 3 |
| | `number` | Any number value |

### TagSize
One of `'small' | 'default' | 'medium' | 'large'`

|  Name   | Type | Value |
| ------- | ---- | ----- |
| small | `string` | Text size: [FONT_SIZE](/docs/utilities/ui-constants#font_size).small <br />Action width: [FONT_SIZE](/docs/utilities/ui-constants#font_size).small * 1.18 <br /> Padding Vertical: [FONT_SIZE](/docs/utilities/ui-constants#font_size).small <br/> Padding Horizontal: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.75 <br/> Padding Vertical: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.25 |
| default | `string` | Text size: [FONT_SIZE](/docs/utilities/ui-constants#font_size).default <br />Action width: [FONT_SIZE](/docs/utilities/ui-constants#font_size).default * 1.18 <br /> Padding Vertical: [FONT_SIZE](/docs/utilities/ui-constants#font_size).default <br/> Padding Horizontal: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.75 <br/> Padding Vertical: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.25 |
| medium | `string` | Text size: [FONT_SIZE](/docs/utilities/ui-constants#font_size).medium <br />Action width: [FONT_SIZE](/docs/utilities/ui-constants#font_size).medium * 1.18 <br /> Padding Vertical: [FONT_SIZE](/docs/utilities/ui-constants#font_size).medium <br/> Padding Horizontal: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.75 <br/> Padding Vertical: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.25 |
| large | `string` | Text size: [FONT_SIZE](/docs/utilities/ui-constants#font_size).large <br />Action width: [FONT_SIZE](/docs/utilities/ui-constants#font_size).large * 1.18 <br /> Padding Vertical: [FONT_SIZE](/docs/utilities/ui-constants#font_size).large <br/> Padding Horizontal: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.75 <br/> Padding Vertical: [BASE_SIZE](/docs/utilities/ui-constants#basesize) * 0.25 |

### TagType 
One of `'default' | 'text' | 'ghost'`

| Name | Type | Description |
| ---- | ---- | ----------- |
| default | `string` | Property to fill in solid mode the background color of the Tag component. |
| ghost | `string` | Property to fill the background with a transparent color the Tag component. |
| text | `string` | Property to fill the background with a transparent color. |

### TagRadius
One of `'none' | 'small' | 'medium' | 'large' | 'full' | number`

| Name | Type | Value |
| ---- | ---- | ----- |
| none | `number` | 0 |
| small | `number` | 2 |
| medium | `number` | 4 |
| large | `number` | 6 |
| full | `number` | 99999 |
| | `number` | Any number value. |
