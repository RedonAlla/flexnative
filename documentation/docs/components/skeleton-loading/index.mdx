The **FlexNative Skeleton Loading** is part of the `FlexNative` and is available under the `@flexnative/skeleton-loading` NPM package.

## Dependencies
- [`@flexnative/theme-context`](../../theme/index.mdx)
- [`react-native-svg`](https://github.com/software-mansion/react-native-svg)

<Installation title="FlexNative Skeleton Loading" name='@flexnative/skeleton-loading' />

## API

```jsx
import SkeletonLoader from '@flexnative/skeleton-loading';
```

## Component

**SkeletonLoader**

Type: <code>React.Component\<[IContentLoaderProps\<T\>](#properties)\></code>

## Properties

:::info
**`IContentLoaderProps`** props extends [`SvgProps`](https://github.com/software-mansion/react-native-svg)
so any [`SvgProps`](https://github.com/software-mansion/react-native-svg) propertie it is applicable also in **`IContentLoaderProps`**
:::

|  Name | Type | Required | Default Value | Description |
| -- | -- | -- | -- | -- |
| animate | `boolean` | **false** | `true` | Boolean indicator whether to animate skeleton loading or not. |
| backgroundColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | `undefined` | Background color. |
| foregroundColor | [`ColorValue`](https://reactnative.dev/docs/colors) | **false** | `undefined` | Foreground color. |
| speed | `number` | **false** | `1.2` | Animation speed. |
| interval | `number` | **false** | `0.25` | Animation interval. |
| uniqueKey | `string` | **false** | `random string` | Used for unique key for `<Rect fill='' clipPath='' />` |
| beforeMask | `JSX.Element` | **false** | `random string` | Costum JSX.Element to render before loading mask. |
