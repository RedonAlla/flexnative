## API

```jsx
import { NumericTextBox } from '@flexnative/inputs';
```

## Component

**NumericTextBox**

Type: <code>React.PureComponent\<[NumberTextBoxProps\<T\>](#properties)\></code>

## Properties

:::info
`NumberTextBoxProps` props extends [`TextInputProps`](/docs/components/inputs/text-box#properties)
so any [`TextInputProps`](/docs/components/inputs/text-box#properties) is applicable also in [**NumberTextBox**](#component) component.
:::

|  Name   | Type | Required | Default Value | Description |
| -- | -- | -- | -- | -- |
| actions | `boolean` | **false** | `true` | Boolean to determinate wether to show or not increment and decrement actions. |
| step | `number` | **false** | `1 ` | Increments and decrement step. |
