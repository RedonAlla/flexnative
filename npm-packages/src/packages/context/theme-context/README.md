FlexNative Theme Context is part of the FlexNative framework and is available under the `@flexnative/theme-context` NPM package.

The theme object is where you define your application's color palette.
By default, `@flexnative/theme-context` provides two color palettes: light and dark.
You can also extend the `BaseTheme` object to add custom or extra colors.

## Installation

```bash
npm install @flexnative/theme-context
```
