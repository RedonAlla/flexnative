# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.6] - 2024-05-22

### Added
- Initial implementation of the `Badge` component.
- Support for multiple variants: `standard`, `outline`, and `dot`.
- Integration with `@flexnative/theme-context` for seamless styling.
- Comprehensive TypeScript definitions.
- Customizable sizes: `small`, `medium`, and `large`.
- Support for numeric and string labels.

### Fixed
- Minor styling inconsistencies across different screen densities.

*Note: Versions prior to 0.0.6 were used for internal development and setup.*
